# 预构建工具索引（已移除）

原先支持通过 `--build-tool-index` 与 `tool-index-build.json` 预先将工具描述写入向量库、供主会话 stage1 参考。**该整条链路已删除**。主会话工具阶段现为 **动态工具**：`ToolCatalogIndex` 本地索引 + `AgentToolingPlugin`（`search_available_tools` / `activate_tools`），见 `backend/Services/DynamicTooling/`。

记忆与知识库仍使用同一套 `IVectorStore` / Embedding；历史 `rag.db` 中若存在 `tools:*` 集合，仅为**不再读取的孤儿数据**，可忽略或手动清理。
