外部 MCP 组件

配置第三方 MCP 服务器（如数据库查询工具），拓展 AI 能力。开关关闭后该组件不会随对话启动。

推荐组件

销售数据库 MCP
只读查询销售用 SQL Server；连接串在 MCP 自身 appsettings 或环境变量 SALES_DB_CONNECTION_STRING 中配置。

添加后可在下方列表中编辑「启动参数」，将项目路径改为本机路径（如 D:\CodeBase\_Taskly\sales-db-mcp\SalesDbMcp.csproj），除非后端从仓库根目录启动。

准确数据
将需要保证准确性的数据以文件形式存于后台目录，供 AI 写入与按 id 读取。已作为内置插件提供，无需在 MCP 列表中添加。数据目录由设置中的「准确数据目录」或配置 AccurateDataDirectory 指定。
定时任务
供 AI 创建与管理定时任务（.task.md + meta）；到点后由后台将任务内容发给 AI 执行。已作为内置插件提供，无需在 MCP 列表中添加。数据目录由设置中的「定时任务目录」或配置 ScheduledTasksDirectory 指定。

由后台规则决定生成的计划是否需要用户确认后再执行。步数不超过阈值且未涉及敏感工具时可直接执行，否则需用户确认。
